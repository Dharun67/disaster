┌────────────────────────────────────────────────────────────────────────────┐
│                        🚀 HOW IT WORKS                                     │
└────────────────────────────────────────────────────────────────────────────┘

USER CLICKS EMERGENCY BUTTON:
1. System detects current location
2. Alert sent to backend
3. Backend notifies:
   - Police stations (5km radius)
   - Hospitals (5km radius)
   - Rescue teams (10km radius)
4. User sees confirmation
5. 5-minute countdown starts
6. Location updates every 5 seconds
7. User can cancel or wait for help

OFFLINE MODE:
1. User clicks emergency button
2. System detects offline mode
3. Alert queued locally
4. User sees "Offline Mode" message
5. When online, alerts automatically sent
6. Or user can manually retry
