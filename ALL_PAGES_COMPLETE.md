## 🎓 How to Use

### Starting the Application
```bash
# Terminal 1: MongoDB
mongod

# Terminal 2: Backend
cd backend && npm start

# Terminal 3: AI Service
cd ai-service && python predictor.py

# Terminal 4: Frontend
cd frontend && npm start
```
