┌────────────────────────────────────────────────────────────────────────────┐
│                        📊 HOW IT WORKS                                     │
└────────────────────────────────────────────────────────────────────────────┘

FIRST TIME USER:
1. Visits app
2. Redirected to login page
3. Enters email
4. Receives OTP
5. Enters OTP
6. Logged in for 30 days
7. Never needs to login again (within 30 days)

RETURNING USER (within 30 days):
1. Visits app
2. Already logged in
3. Directly goes to dashboard
4. No need to login again

RETURNING USER (after 30 days):
1. Visits app
2. Session expired
3. Redirected to login
4. Must login again

BEFORE SESSION EXPIRY:
1. 5 minutes before expiry
2. Warning modal appears
3. User can extend session
4. Session extended for 30 more days

│                        🔧 CONFIGURATION                                    │
└────────────────────────────────────────────────────────────────────────────┘

CHANGE SESSION DURATION:
Edit authService.js:
  authService.setToken(token, 30); // Change 30 to desired days

CHANGE OTP EXPIRY:
Edit backend server.js:
  const otpExpiry = new Date(Date.now() + 10 * 60 * 1000); // 10 minutes

CHANGE WARNING TIME:
Edit SessionManager.js:
  if (timeRemaining === 5) { // Change 5 to desired minutes
    setSessionWarning(true);
  }
