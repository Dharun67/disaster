## 📊 Data Flow Diagram

### Featured Page
```
User Login
    ↓
Navigate to /featured
    ↓
Component Mounts
    ↓
fetchFeaturedResources()
    ↓
API Call: GET /api/featured-resources
    ↓
Data Received (or Mock Data)
    ↓
State Updated
    ↓
Render Featured Grid
    ↓
User Interactions:
├─ Filter by Category
├─ View Ratings
├─ Check Verified Badge
└─ See Impact Metrics
```

### Marketplace Page
```
User Login
    ↓
Navigate to /marketplace
    ↓
Component Mounts
    ↓
fetchMarketplaceProducts()
    ↓
API Call: GET /api/marketplace-products
    ↓
Data Received (or Mock Data)
    ↓
State Updated
    ↓
Render Product Grid
    ↓
User Interactions:
├─ Search Products
├─ Filter by Category
├─ Sort by Price/Rating
├─ Add to Cart
└─ View Total Price
```

## 🎓 Key Concepts Visualized

### Filtering Logic
```
All Products (9)
    ↓
Category Filter
    ├─ Water (2)
    ├─ Medical (1)
    ├─ Lighting (1)
    ├─ Survival (1)
    ├─ Communication (1)
    ├─ Rescue (1)
    ├─ Food (1)

```

### Sorting Logic
```
Products
    ↓
Sort By:
├─ Popular (by reviews)
├─ Price Low to High
├─ Price High to Low
└─ Highest Rated
```

### Cart Calculation
```
Product 1: ₹2,499 × 1 = ₹2,499
Product 2: ₹1,899 × 2 = ₹3,798
Product 3: ₹599 × 1 = ₹599
                    ─────────
Total:              ₹6,896
```
